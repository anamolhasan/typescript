## Typescript Setup

### first create npm package
```bash
npm init -y
```

### global install 
```bash  
npm install -g typescript  
```
### local install 
```bash
 npm install typescript --save-dev  
 ```
### Create tsconfig.json  file 
```bash
 tsc --init  
 ```
### typescript file compile
```bash
tsc
```
### typescript file run
```bash
node dist/index.js
```
